# Scoreboard
A TypeScript application
## Running
To run the app, enter
```
npm start
```
in the terminal.
## Base URL
You have to set baseURL in app/controller/user.controller.ts to the URL of the server. This is used for E-Mail verification.
```ts
const baseURL = 'http://localhost:8080'
```
